# Contributing

We are open to, and grateful for, any contributions made by the community.

## Requirements

This repository requires using [Bun](https://bun.sh/) as your runtime.

## Local Development

You can run a local development server by running `bun run start:dev` which will monitor for any changes and rebuild the examples on every code change.

## Building

To prepare a new release run `bun dist` which will generate the necessary output files in `/dist`.
